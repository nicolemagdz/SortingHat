What is an algorithm?

Process designed to solve a problem

Set of instructions that ends up in a desired state

How does an algorithm work?

  • Input: Receiving user data
  • Processing: Applying calculations, learned patterns, rules
  • Decision: Selecting best outcome
  • Output: Final decision is produced

Types of algorithms

Rule-based Algorithms

Machine Learning Algorithms

Optimization Algorithms

The Sorting Hat as an algorithm

Sorting Hat "looks" into a student's mind

Considers personality traits, values, preferences

Sometimes allows user input

Results in a quick and efficient, yet considerate decision for house placement
